Input Help

Multiple High Schools/Majors can be selected at once by seperating them via commas.

  • Example input: Accounting, Bio-Chemistry

SAT's range from 0 to 1600 and do not include Writing. GPA ranges from 0.0 to 4.0. Multiple SAT/GPA ranges can be added with commas and dashes.

  • Example SAT input: 3-3.2, 4
  • Example GPA input: 1000, 1440-1500
